# GOOD Documentation Example

# HabitTracker — Quick Start Guide

## What This App Does
HabitTracker helps users build daily habits (e.g. drinking water, running, meditation) by:
- Creating habits
- Marking them as completed each day
- Tracking progress over time

---

## How to Log a Habit (Step-by-Step)

1. Open the app  
2. Navigate to the **“My Habits”** screen  
3. Click **“+ Add Habit”** (if the habit does not exist)  
4. Select a habit  
5. Click **“Mark as Complete”**

You should now see your progress updated

---

## Common Confusion

- **“Nothing happens when I press the button”**  
  - Make sure a habit is selected before marking it complete

- **“My progress reset”**  
  - The app tracks activity **per day**, not cumulative completion

---

## Why It Works This Way

- Habits reset daily to encourage **consistent repetition**
- This avoids “bulk completion” and supports behaviour change

---

## System Overview (High-Level)

- Users interact through a simple UI
- The app stores:
  - Habit name
  - Completion status per day
- Progress is calculated based on daily completion frequency

---

## Key Takeaways

- You must complete habits **daily**
- Progress is based on **consistency**, not total actions
- The system does **not auto-complete habits**
