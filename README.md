# ELEC1005 V&V Lab

⚠️ **Important**  
The app will NOT work immediately after import.
This is intentional.

Your task is to connect it to your own SharePoint list and fix any issues you encounter.

---

## 1. Files in This Repository

- `app/ELEC005-2026-S1-W10.msapp` — the Power Apps application
- `README.md` — setup instructions (this file)

---

## 2. Prerequisites

- Access to Power Apps
- Access to SharePoint Online
- Permission to create SharePoint lists

---

## 3. Step 1: Create Your SharePoint List

Create a **new SharePoint list** as instructed in Lab 6 or use your previously created list.

---

## 4. Step 2: Import the App

1. Go to **Power Apps**
2. Select **Apps**
3. Choose **Import canvas app**
4. Upload `ELEC005-2026-S1-W10.msapp`

---

## 5. Step 3: Connect the App to Your List

After importing:

1. Open the app in **Edit mode**
2. Go to **Data**
3. Add your SharePoint list as a data source
4. Remove or fix any broken data source references

You should expect errors at this stage.

---

## 6. Debugging Tasks (Expected)

Continue breaking the software as instructed in the lab manual.

---

Good luck — and break things on purpose 🙃
